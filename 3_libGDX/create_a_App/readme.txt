1. Open a control line (e.g. Git Bash)

2. go to the Download Folder

3. tab the following in:
	java -jar ./jdx-setup.jar

4. Add your settings
	Name: <App name>
	Package: <Package from Android Studio> [something like: com.basbaer.<app name>
	Game class: <how you want the class to be called>
	
	Android SDK: -> do the following:
		- go to Android Studio
		- Tools > SDK Manager
		- copy the 'SDK Location'
	
	Sub Projects:
	-> select only the areas you want to build the app in

	Extensions:
	-> stay with the selected options

5. 'Generate'

6. Open the App in Android Studio like a Android Studio Project
